Action Items:

  • Download, Install and open Xcode 14. Install Command Line Tools when directed.
  • Make sure you have an Apple ID and sign in to Apple Developer with it.
  • Have a GitHub account.

Xcode Releases is a handy site that organizes download links to versions of Xcode.

I mentioned the Swift Playgrounds app for iPad, but I forgot that there’s a Mac version too! This is a great way to get started with Swift, so check that out.
